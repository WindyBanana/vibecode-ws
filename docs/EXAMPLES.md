# Example Projects

Get inspired! Here are things you can build with this template.

## Landing Page

**What it is:** A simple website to promote your business, product, or service.

**Tell your AI:**
```
"Create a landing page for my bakery with:
- Hero section with my logo
- Features section (3 features)
- Contact form
- Footer with social links"
```

**What you'll get:**
- Beautiful, responsive landing page
- Contact form that sends emails
- Mobile-friendly design
- Ready to deploy

**Time:** 30-60 minutes

## Tips for Building

### Start Simple
Build the basics first, then add features:

```
"Build a simple blog"
[Test it]

"Add categories to the blog"
[Test it]

"Add search functionality"
[Test it]
```

### Iterate
Don't try to build everything at once:

```
"Create a landing page"
[See it, decide what to change]

"Make the buttons blue"
"Add more spacing"
"Use a different font"
```

### Ask for Examples
Your AI can show you similar projects:

```
"Show me examples of good portfolio sites"
"What features should a recipe site have?"
```
